### [189. 轮转数组](https://leetcode.cn/problems/rotate-array/)

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

 

**示例 1:**

```
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]
```

**示例 2:**

```
输入：nums = [-1,-100,3,99], k = 2
输出：[3,99,-1,-100]
解释: 
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]
```

 

**提示：**

- `1 <= nums.length <= 105`
- `-231 <= nums[i] <= 231 - 1`
- `0 <= k <= 105`

 

**进阶：**

- 尽可能想出更多的解决方案，至少有 **三种** 不同的方法可以解决这个问题。
- 你可以使用空间复杂度为 `O(1)` 的 **原地** 算法解决这个问题吗？



**题解**

方法一：切片法

对于示例一

```text
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]
```

可以把`nums`数组的后`k `个元素进行切片，移植到`nums` 的前部

```python
k = k % len(nums)
slice = nums[-k:]
del nums[-k:]
nums[0:0] = slice #在第 0 个位置之前插入
```

时空复杂度:`O(n)`

方法二：移动法

```python
for _ in range(k):
	nums.insert(0, nums.pop())
```

方法三：三次反转法

可以用某种“整体操作”来完成移动，而不是逐个处理。一种经典的方法是**反转**，因为反转操作可以在原地完成，且时间复杂度低。

**右边**：nums[::-1] 生成一个反转后的新列表。

start：开始索引（包含，默认 0）。

end：结束索引（不包含，默认到列表末尾）。

step：步长（默认 1，负数表示反向）

**左边**：nums[:] 表示将这个新列表赋值给 nums 的整个范围。

```python
nums[:] = nums[::-1]        # 反转整个列表
nums[:k] = nums[:k][::-1]   # 反转前 k 个
nums[k:] = nums[k:][::-1]   # 反转后 n-k 个
```
